# Currex Blog CMS

A modern, self-hosted blog CMS with admin panel and static site generator.

---

## Features

- Admin Panel with dashboard and full CRUD
- Quill.js rich text editor
- Custom menus and sidebar widgets
- Unlimited pages (About, Contact, etc.)
- Featured images and auto-excerpt
- Static site generator for GitHub Pages
- Markdown storage, no database needed
- Responsive design, SEO ready

---

## Quick Start

1. Install Node.js 18+
2. git clone ... && cd currex-blog
3. npm install
4. npm start
5. Open http://localhost:4000

---

## Usage

- Admin panel at /admin (admin / admin123)
- Posts: Write blog posts with rich text editor
- Pages: Create custom pages (About, etc.)
- Menus: Customize navigation
- Sidebar: Configure widgets
- Settings: Site title, social links
- Build: Generate static site

---
